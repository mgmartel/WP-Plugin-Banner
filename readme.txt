Show WP.org repository banners on your website using a shortcode

== Description ==

Show plugin banners from WP or your own website by using a simple shortcode.

= Usage =

To get the image from WP.org:

`[plugin-banner name="Plugin Name" slug="plugin-slug"]`

With your own image:

`[plugin-banner name="Plugin Name" img="http://image-url/"]`
